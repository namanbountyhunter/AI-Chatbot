This project is a full-stack AI-powered chatbot application designed to deliver intelligent, context-aware conversations through a seamless web interface. The system integrates a React-based frontend with a Django backend, enabling efficient handling of user queries, session management, and real-time interaction with an AI model.

The backend architecture is modular, with dedicated services for handling chat logic, maintaining conversation history, and interacting with the AI engine. Features such as session-based chat storage, message history optimization, and API-driven communication ensure scalability and performance. The chatbot is capable of maintaining contextual continuity by retrieving recent interactions, improving response relevance.

On the frontend, a dynamic and responsive user interface allows users to send messages, view conversation threads, and manage chat sessions. Additional functionalities like chat deletion, loading states, and clean UI interactions enhance user experience.

The project also includes image upload support, enabling multimedia interaction, and is designed with extensibility in mind for future enhancements like voice input, authentication, and deployment.
